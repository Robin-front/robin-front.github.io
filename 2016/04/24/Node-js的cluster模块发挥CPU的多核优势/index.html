Node.js的cluster模块发挥CPU的多核优势

发表于 2016-04-24 | 更新于 2016-04-24 | 分类于 Node |
| 字数统计 73 | 阅读时长 1

一个Node.js应用程序只能使用在一个线程中。为了充分发挥多核CPU的优势，Node.js中提供了一个 cluster模块，允许在多个子进程中运行不同的Node.js应用程序。cluster模块是对 child_process.fork()的一个封装。
