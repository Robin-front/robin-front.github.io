pm2一个带有负载均衡功能的Node应用的进程管理器

发表于 2016-05-04   |   分类于 Node   |  

PM2是一个提供了内置的负载均衡的Node.js应用程序进程管理器。它可以让你保持应用程序永远在线，无需停机地重启应用，也有助于系统管理员工作。
